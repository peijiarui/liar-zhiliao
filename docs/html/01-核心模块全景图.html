企业级 RAG 知识库 - 核心模块全景图

一个完整的企业级 RAG 知识库通常包含以下六大核心模块。请先浏览整体架构，我们在后续逐步细化每个模块的技术选型。

六大核心模块

1. 文档摄入层

多格式文档解析（PDF/Word/Markdown/HTML/图片）、文档分割策略、元数据提取、OCR 识别

关键能力：格式兼容性 + 解析精度

2. 向量化与索引层

Embedding 模型选型、向量数据库、稀疏/稠密混合检索、索引构建策略

关键能力：语义理解 + 检索精度

3. 检索层

混合检索（Hybrid Search）、Rerank 重排序、多路召回融合、查询改写/扩展

关键能力：召回率 + 准确率

4. 生成层

LLM 推理、上下文构建、引用溯源、工具调用（Function Calling）、流式输出

关键能力：答案质量 + 可追溯性

5. 治理与安全层

权限控制（RBAC/ABAC）、数据脱敏、审计日志、内容安全审核、限流熔断

关键能力：安全合规 + 可审计

6. 平台与运营层

知识库管理后台、对话管理、用户管理、监控告警、效果评估、反馈闭环

关键能力：可运营 + 可观测
