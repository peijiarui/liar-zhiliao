统一 ETL 处理管道

→ 🖼️ 图片 OCR（Tesseract） → 🧮 Embedding 向量化 → 💾 写入 Milvus + ES + MinIO

文档分割策略

递归字符分割

按段落→句子→字符优先级递归切分，保留语义完整性。适合大多数文档。

LangChain4j: DocumentSplitter

语义分割

基于 Embedding 相似度断点切分，句子间语义差异大时断开。精度高但计算成本大。

适合长文档/论文

父子文档

大块（Parent）做完整上下文，小块（Child）做精确检索。Rerank 时引用 Parent 还原上下文。

推荐！兼顾精度与上下文

关键技术选型

文档解析Apache Tika 2.x支持 1000+ 格式，Java 生态最佳选择
OCR 识别Tesseract 5 + PDFBox扫描件 PDF / 图片文字提取
表格处理Apache POI + 自定义策略Excel 需保留行列结构信息
增量同步定时任务 + 文件指纹 (MD5)避免重复处理，追踪文档变更
异步处理消息队列 (RabbitMQ)大文档异步解析，避免阻塞
