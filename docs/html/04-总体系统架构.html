总体系统架构

基于 Spring Boot + LangChain4j 的企业级高级 RAG 知识库

系统架构分层图
接入层
Web UI (React) REST API SSE 流式 WebSocket
网关与安全层
Spring Security + OAuth2 SSO RBAC 权限 多租户隔离 Rate Limiter 审计日志
业务服务层
文档管理服务
上传/解析/分割/版本管理
检索服务
查询改写/混合检索/Rerank
对话服务
LLM编排/流式输出/引用溯源
知识库管理
多知识库/权限/分类/标签
用户与租户
SSO/部门/角色/权限
运营监控
指标采集/告警/效果评估
AI 编排层 (LangChain4j)
@AiService EmbeddingStore ContentRetriever ChatMemory Tool (@Tool)
数据存储层
Milvus (向量) Elasticsearch (关键词) PostgreSQL (业务) Redis (会话/缓存) MinIO (文档文件) RabbitMQ (异步任务)

模块依赖关系

Maven 模块拆分（推荐）

liar-rag/
├── liar-rag-common      (公共模型/工具)
├── liar-rag-ingestion   (文档摄入)
├── liar-rag-retrieval   (检索服务)
├── liar-rag-chat        (对话服务)
├── liar-rag-auth        (认证授权)
├── liar-rag-admin       (管理后台)
└── liar-rag-app         (启动入口，组装)
